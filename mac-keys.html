Mac 快捷键总结

2018/1/8 收录于   经验

之前买了一个键盘膜，上边有输入法提醒，感觉蛮不错了（嘿嘿，我就是看见有快捷键才买的，快捷键总结之后，膜就再也没用过 😂）
此博客持续更新

  • 全屏截图：command + shift + 3
  • 局部截图：command + Shift + 4
  • 彻底清除：command + delete
  • 清空废纸篓：command + Shift + delete
  • 切换程序：command + tab
  • 切换上一个程序：command + Shift + tab
  • 退出程序：command + q
  • 注销用户：command + Shift + q
  • 关闭窗口：command + option + w
  • 打开Airdrop：command + Shift + r
  • Finder中新建标签：command + t
  • Finder中打开实用工具：command + Shift + u
  • Finder中显示文件简介：command + i
  • Finder中打开iCloudDrive：command + Shift + i
  • Finder中打开文件：command + o
  • Finder中打开下载：command + option + l
  • 打印文件：command + p
  • 后退：command + [
  • 前进：command + ]
  • 隐藏前端端口：command + h
  • 隐藏其他窗口：command + option + h
  • 打开网络：command + Shift + k
  • 打开链接服务器：command + k
  • 打开计算机：command + Shift + c
  • Safari 隐藏书签栏：command + Shift + b
  • 偏好设置：command + ，
  • 获取帮助：command + Shift + /
  • 打开spotlight搜索：command + 空格
  • 打开Finder搜索：command + option + 空格
  • 中英文切换：caps lock（大小写切换键）
  • 切换输入法：control + 空格
