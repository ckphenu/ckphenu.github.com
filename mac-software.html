Mac 应用整理

2018/1/4 收录于   经验

大部分软件都非官方版本，请慎重，你懂的
我的原则是能用官方提供的就尽量不找第三方，不是因为官方是最好的，而是不愿意装太多相同性质的软件
如果下载链接失效，请自行搜索
推荐网站：http://xclient.info/

软件开发类

  • Intellij IDEA 公认的Java开发工具，社区版是免费，功能基本够用，当然你要是有情怀可以支持一下商业版，如果你要是屌丝，请点击这里。
  • VS Code 特别适合前端开发，也可以作为文本编辑器和markdown编辑器用，支持各种插件
  • SourceTree 可以理解为Git图形化工具，非常好用
  • Postman用与模拟Ajax请求，可以作为restfull API的测试工具，功能相当强大
  • Navicat Premium 一款功能超全的数据库客户端
  • DBeaver 数据库客户端，支持H2等内存数据库的连接
  • ShellCraft强大的SSH/Telnet客户端
  • Transmit功能齐全的FTP客户端
  • Redis Desktop Manager Redis 桌面应用程序

辅助使用类

  • Cheatsheet 快捷键提醒，长按 command 键会弹出当前焦点软件的全部快捷键
  • 百度云下载插件
  • 欧路词典
  • HandShaker

  • 微软远程连接
  • Zoom.us 远程会议
