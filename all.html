Mac 快捷键总结

之前买了一个键盘膜，上边有输入法提醒，感觉蛮不错了（嘿嘿，我就是看见有输入法才买的，输入法总结之后，膜就再也没用过 😂）

  • 全屏截图：command + shift + 3
  • 局部截图：command + Shift + 4
  • 彻底清除：command + delete
  • 清空废纸篓：command + Shift + delete
  • 切换程序：command + tab
  • 切换上一个程序：command + Shift + tab
  • 退出程序：command + q
  • 注销用户：command + Shift + q
  • 关闭窗口：command + option + w
  • 打开Airdrop：command + Shift + r
  • Finder中新建标签：command + t
  • Finder中打开实用工具：command + Shift + u
  • Finder中显示文件简介：command + i
  • Finder中打开iCloudDrive：command + Shift + i
  • Finder中打开文件：command + o
  • Finder中打开下载：command + option + l
  • 打印文件：command + p
  • 后退：command + [
  • 前进：command + ]
  • 隐藏前端端口：command + h
  • 隐藏其他窗口：command + option + h
  • 打开网络：command + Shift + k
  • 打开链接服务器：command + k
  • 打开计算机：command + Shift + c
  • Safari 隐藏书签栏：command + Shift + b
  • 偏好设置：command + ，
  • 获取帮助：command + Shift + /
  • 打开spotlight搜索：command + 空格
  • 打开Finder搜索：command + option + 空格
  • 中英文切换：caps lock（大小写切换键）
  • 切换输入法：control + 空格

碎碎念

整理生活中稍纵即逝的想法

  • 工作就像购物，只不过你购买的不是物品而是信誉，你消费的也不是金钱而是别人对你的信任
  • 前一段时间广播里说十年里最大的变化是什么，最近朋友圈又在疯狂的晒18岁，十年前差不多也是我的18岁，这个问题让我陷入了深深的思考，不是想最大的变化，而是想，那个时候正好是无忧无虑到步入社会的转折，幸运的是在十年中每一个人生的重要节点，都能遇到那么一两个人，给我指点迷津，帮我度过难关，这些人有些是家人、亲戚、兄弟、朋友、领导、同事，也许他们中间的某些人只陪我们走一过了人生的一小段，但我们依然应该心存感激，感激他们曾经给你的某大帮助。
  • 有时候我经常反思，为什么有些人做什么事情都能游刃有余，有些人总是能风风光光叱咤风云……，那是因为在我们熬夜玩游戏的时候，在我们聚会嗨的时候，他一个人孤独的在求知，对于一个平凡人来说，只有孤独的人，才能真正静下来思考人生，才能回味过往，规划未来

Mac 应用整理

大部分软件都非官方版本，请慎重，你懂的
我的原则是能用官方提供的就尽量不找第三方，不是因为官方是最好的，而是不愿意装太多相同性质的软件
如果下载链接失效，请自行搜索
推荐网站：http://xclient.info/
推荐网站：https://nicoapp.pipipan.com/dir/13311048-20509288-95d6dc/24612944/

软件开发类

  • Intellij IDEA 公认的Java开发工具，社区版是免费，功能基本够用，当然你要是有情怀可以支持一下商业版，如果你要是屌丝，请点击这里。
  • VS Code 特别适合前端开发，也可以作为文本编辑器和markdown编辑器用，支持各种插件
  • SourceTree 可以理解为Git图形化工具，非常好用
  • Postman用与模拟Ajax请求，可以作为restfull API的测试工具，功能相当强大
  • Navicat Premium 一款功能超全的数据库客户端
  • DBeaver 数据库客户端，支持H2等内存数据库的连接
  • ShellCraft强大的SSH/Telnet客户端
  • Transmit功能齐全的FTP客户端
  • Redis Desktop Manager Redis 桌面应用程序

辅助使用类

  • Cheatsheet 快捷键提醒，长按 command 键会弹出当前焦点软件的全部快捷键
  • 百度云下载插件【Aria2GUI】附上使用教程
  • 欧路词典专为MAC而生的词典
  • HandShaker Mac 电脑上也可以方便自如地管理 Android 手机中的内容，锤子给我最大的惊喜

利用mWeb 编辑博客

第一步肯定是先去官网下载安装文件了，当然你要是一个穷屌丝，又没有情怀，请根据国情自行安装软件。
言归正传，我们需要先创建一个分类，启动APP，点击在左下角的 “➕” 号，选择【新增分类】，弹出如下框,填写相应的名称，选择【设置为静态网站】

会弹出如下页面，根据需要进行相应的设置。

关于生成路径的设置：clone Github的仓库默认是新创建的目录，所以个人喜好在，生成位置设置的目录里面，新建目录用来clone GitHub，并将刚才网站的名称指定为刚才创建的目录名称。这样在执行【生成网站】的时候就会把静态文件直接生成到GitHub目录，就可以直接push到服务了。

进行完简要的设置之后，我们就可以编写第一个遍博客了。
